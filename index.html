# 隐私权政策 本隐私权政策适用于 Chrome 浏览器扩展「StrangeHighlight - 生词高亮」（以下简称“本扩展”）。 ## 1. 数据收集与使用 本扩展严格保护用户的隐私安全，不会收集、上传、存储、分享任何用户的个人身份信息、敏感信息，也不会将任何用户数据传输至第三方服务器。 本扩展仅会在用户的本地设备上，通过浏览器的本地存储能力，存储用户手动添加的生词、对应的注释内容，以及扩展的基础配置信息。所有数据均仅保存在用户自己的设备中，开发者无法访问、获取这些数据。 ## 2. 权限使用说明 本扩展申请的所有权限，均仅用于实现扩展的核心功能，具体如下： - storage权限：仅用于在用户本地设备持久化存储生词、注释与配置数据； - activeTab权限：仅在用户主动打开网页时，在当前激活的标签页中识别生词、渲染高亮样式，不会后台读取网页内容； - contextMenus权限：仅用于为用户提供右键快捷添加生词的操作入口； - 主机权限：仅用于支持用户在任意网页中使用扩展的核心功能，不会收集用户的网页浏览数据。 ## 3. 数据安全 本扩展所有数据均存储在用户本地设备的浏览器沙箱中，不会通过网络传输，不存在数据泄露、被第三方获取的风险。用户可随时通过扩展的功能界面，删除本地存储的所有生词数据。 ## 4. 政策更新 如本扩展的功能发生重大变更，导致隐私权政策需要调整，我们会通过扩展更新说明、本页面更新的方式告知用户。本政策更新后，将以更新后的内容为准。 ## 5. 联系我们 如您对本隐私权政策有任何疑问、建议，可通过邮箱 m15016190267@gmail.com 联系我们。
